# User Management API

Simple ASP.NET Core Web API for managing users.

## Features
- CRUD operations (Create, Read, Update, Delete)
- Input validation
- Custom logging middleware
- InMemory database
- Swagger UI

## Endpoints
- GET /users
- GET /users/{id}
- POST /users
- PUT /users/{id}
- DELETE /users/{id}

## Tools Used
- ASP.NET Core Web API
- Entity Framework Core (InMemory)
- GitHub Copilot (for code suggestions and debugging)

## Run Project
```bash
dotnet run
